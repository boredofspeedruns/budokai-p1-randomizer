# Dragon Ball Z: Budokai — Player 1 Randomizer (All Modes)

This mod randomizes **Player 1's character selection** across all game modes and optionally enables each character's **Breakthrough Capsule** automatically.

## Features

### Cheat 1 — Player 1 Character Randomizer
- Randomizes Player 1's character in all modes.

### Cheat 2 — Automatic Breakthrough Capsules
- Automatically assigns the selected character's Breakthrough Capsule when using the randomizer.

---

## Notes

- Alternate costumes are used where possible to prevent the common **vertex corruption issues** that can occur during mirror matchups.
- The only exceptions are:
  - **Saibaman**
  - **Cell Jr.**

**Saibaman and Cell Jr.** can still use their proper movesets with the Breakthrough cheat enabled by equipping **Goku's Breakthrough Capsule**.

This allows them to retain:
- Grabs
- Acid
- Kamehameha
